Hotel Management API

A practice project built with Node.js, Express, and MongoDB to manage hotel operations.
It provides CRUD (Create, Read, Update, Delete) APIs for managing hotel staff (Person) and menu items.

🚀 Features

👨‍🍳 Person Management → Add, update, view, and delete hotel staff (Chef, Waiter, Manager).

🍴 Menu Management → Add, update, view, and delete menu items.

🗄️ MongoDB Integration → Stores all staff and menu details.

⚡ REST API with Express → Clean and structured API endpoints.

✅ Validation with Mongoose → Ensures correct data handling.

🛠️ Test APIs → Using cURL, Postman, or Hoppscotch.
